

# 🐍 Python Basics Jupyter Notebook

This repository contains basic Python programs created using Jupyter Notebook.  
It is designed for beginners to understand fundamental concepts of Python.

---

## 📌 Topics Covered
- Arithmetic Operations
- Variables and Data Types
- Basic Calculations
- Expressions and Output Formatting

---

## 💻 Programs Included
1. Addition and Subtraction
2. Variable Operations
3. Division Calculation
4. Rectangle Perimeter
5. Total Cost Calculation
6. Square of a Number
7. Multi-line Comments
8. Personal Information Program
9. Expression Evaluation

---

## 🚀 How to Run
1. Open the notebook in Jupyter Notebook / VS Code
2. Run each cell step by step
